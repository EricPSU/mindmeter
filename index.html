Lap Distance
The lap distance establishes where on the track you will capture your splits. This will also define the number of laps.
Partial Lap
For events that have laps that don't go the full 400 meters, such as the 1500 and 3000. Define if you want the 300 meter "partial lap" to be captured as the first or last lap. The lap pace for the partial lap is increased as if it were 400m.
Target Time
:
Establishing a target finish time will display a target lap split that updates based on the pace of the previous lap. Press clear if you want to remove this setting value.
